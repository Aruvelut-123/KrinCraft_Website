# 常见问题

## 注册
第一次进入服务器需先注册，命令格式：

`/register <密码> <重复密码>`

示例：

`/register 123456789 123456789`

此示例中密码为 `123456789`。

## 如何重置密码
前提：你已在服务器中使用过 `/email add <你的邮箱>` 绑定邮箱；否则请联系管理员。

重置步骤：

1. 在游戏内输入：

`/email recover <你绑定的邮箱>`

2. 请检查你邮箱中的邮件，按邮件中的提示执行命令，例如：

`/email code efccfff3`

3. 然后设置新密码：

`/email setpassword <新密码>`

完成以上步骤后，即可使用新密码登录。

## 登录
注册完成后使用：

`/login <密码>`

示例：

`/login 123456789`
