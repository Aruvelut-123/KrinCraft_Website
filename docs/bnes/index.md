# 🎮街机介绍
这是一个nes模拟插件实现的一个功能，它可以让你在服务器里游玩 超级玛丽，坦克大战，等老游戏！

## 指令
/bnes create <实例名>
输入这个可以创建游戏机，实例名自己想一个，要记住它，输入此指令将会扣除你1000K币，进行购买游戏机。

## 使用
输入/bnes create <实例名> 这个指令后，身上会多出4张地图，找一面墙，放上四个物品展示框，例如这样

![bnes图标1](https://www.onekrincraft.cn/images/bnes.png "bnes图标1")

然后分别放上身上的四张地图，在做两个楼梯，放在地图面前，然后你在1服主城找一个大房子，房子里面有箱子商店，可以购买游戏卡带，购买后，回到游戏机前，右键楼梯或/sit，坐上楼梯再拿游戏卡带右键地图，再空手右键，你就加入了游戏。

|游戏内实际操作|映射按键|
|----|----|
|游戏移动方向|手柄方向|
|跳跃|手柄A|
|交换副手|手柄B|
|左键地图|select键|
|右键地图|start键|
|“跳跃”和“右键地图”同时按下|重置游戏|
