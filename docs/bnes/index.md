# 🎮街机介绍

这是一个 NES 模拟插件功能，让你在服务器里游玩超级玛丽、坦克大战等经典游戏！

## 购买游戏机

使用 `/bnes create <实例名>` 创建游戏机（需要 1000K币）。实例名自己设定并记住。

## 安装与游玩

### 第一步：安装游戏机

1. 执行 `/bnes create <实例名>` 后，背包中会出现 4 张地图
2. 找一面墙，放上 4 个物品展示框，排列如下：

![bnes图标](https://www.onekrincraft.cn/images/bnes.png)

3. 将 4 张地图分别放入展示框中
4. 在地图前放置 2 个楼梯

### 第二步：购买游戏卡带

1. 前往 1 服主城的大房子
2. 在房子里的箱子商店购买游戏卡带
3. 返回游戏机前

### 第三步：开始游戏

1. 坐在楼梯上（右键楼梯或输入 `/sit`）
2. 拿起游戏卡带，右键地图
3. 空手右键地图，即可进入游戏

## 游戏操作

|游戏内实际操作|映射按键|
|--------|--------|
|游戏移动方向|手柄方向|
|跳跃|手柄A|
|交换副手|手柄B|
|左键地图|select键|
|右键地图|start键|
|“跳跃”和“右键地图”同时按下|重置游戏|
