# 💰货币介绍

货币是 KrinCraft 服务器中的重要功能。在1服主城中有许多箱子商店，分为出售和收购两种。

::: tip 提示
2服商店已有完整提示，直接按照指引操作即可。
:::

## 货币指令

| 指令 | 说明 |
|------|------|
| `/money` | 查看你当前的余额 |
| `/baltop` | 查看谁是本服最富有的玩家 |

## 购买（出售）

1. **左键点击** 箱子商店的牌子
2. **输入** 要购买的数量
3. 若余额充足，**自动扣款** 并将物品放入背包
4. 若余额不足则购买失败

## 出售（收购）

1. **左键点击** 收购牌子
2. **输入** 要出售的数量
3. 若背包有相应物品，**自动出售** 并获得余额
4. 若物品不足则出售失败
