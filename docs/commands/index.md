# 🗨️指令介绍

## 🙂基本指令介绍

### 拥有Minecraft正版账号

在服务器大厅使用 `/prem` 指令打开正版验证。开启后无需输入密码即可登录。

::: warning 警告
若你不是正版账号，开启正版验证后将永远无法登入用这个名字进入服务器！(除非管理员将你移除正版验证名单)
::: 

### 打开服务器菜单

使用 `/cd` 打开服务器菜单。


### Home 传送系统

| 指令 | 说明 |
|------|------|
| `/sethome` | 设置默认家的位置（默认名字为 home） |
| `/sethome <家的名字>` | 设置自定义名字的家 |
| `/home` | 传送到默认的 home 位置 |
| `/homes` | 打开 home 管理界面 |

### 返回上一个位置

使用 `/back` 返回到上一个地点。

### 传送功能

| 指令 | 说明 |
|------|------|
| `/tpa <玩家名>` | 请求传送至该玩家身边 |
| `/tpahere <玩家名>` | 请求该玩家传送到你的位置 |
| `/rtp` | 随机传送（有冷却时间） |

### 飞行充能系统

| 指令 | 说明 |
|------|------|
| `/flyc` | 开启或关闭飞行充能 |
| `/flightcharge` | 使用 K币 或经验值充能（消耗点数根据飞行距离计算） |

### 领地系统

详见 [领地使用指南](https://dominion.lunadeer.cn/notes/doc/player/dominion/create/)

::: tip 提示
圈地时使用 **箭** 而非木锄头
:::
