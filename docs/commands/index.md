# 🗨️指令介绍

#### 拥有Minecraft正版账号
可以在服务器大厅输入/prem，之后再次输入/prem

打开正版验证，开启正版验证后你将无需再输入密码登录

::: warning 警告！！
若你不是正版账号开启了正版验证你将无法再登入服务器！
::: 

#### 打开服务器菜单
服务器有1服 2服，其菜单打开指令不同

1服打开的指令为/cd

2服打开的指令为/mc

#### 如何使用home？
/sethome
设置一个默认的家，默认的名字为home

/sethome <家的名字>
自定义home名字
/home 
传送到默认名字为home的位置

/homes
打开home管理界面

#### 如何使用返回上一个地方？
/back
返回到上一个保存的地方

#### 如何使用tpa？
/tpa <玩家名>
 请求传送至该玩家身边

/tpahere <玩家名>
请求该玩家传送到自己所站位置

#### 变身生物
/morph
可变为其它生物(已经击杀过的生物)

/unmorph
可以解除变身
