### 创建领地
#### 条件
一个合法的待创建领地需要满足以下几个条件：
●没有与之重名的领地
●领地区域不与任何已有的领地冲突
#### 过程
选区
创建领地时，你需要指定你领地的边界，以此来确定你领地的范围。使用小木锄选择领地的2个对角点：主手持用木锄时，你可以对第一个方块点击左键，然后得到如下图所示的提示。然后在对第二个方块右键形成一个长方形的领地，也会得到如下图所示的提示。两个方块选完之后，你还会得到如下图所示的额外的提示，其中标绿的字段则是创建此领地所需消耗的K币。在此之后你还可以通过/res select expand [数量] 或 /res select contract [数量]来扩大或缩小选区。
#### 创建
在选择完成后就可以使用/res create [领地名]来创建你的领地了！创建成功后会得到如下图所示的提示。你也可以使用/res auto [领地名] <半径>来自动以你为中心创建指定半径的领地，当然，半径是可选的，如果不填则会根据你的K币余额来自动创建对应大小的领地哦！

#### 管理领地
当你需要修改别人拥有的你的领地的权限时，你可以来看看这里！
注：false为关闭的意思，而true为开启的意思。

#### 禁止TP
首先，使用/res set tp false可以关闭别人tp到你的领地的权限，这对于隐藏你的领地很有帮助。

#### 禁止进入
如果需要防止其他玩家进入你的领地，可以使用/res set move false来防止玩家进入或传送进入你的领地。

#### 朋友权限设置
而如果你有朋友想要给他你领地的权限，也很简单，在你的领地内使用/res padd [玩家名]既可给予对应玩家对你领地的权限！如果你和他不再是朋友了，也没问题！使用/res pdel [玩家名]即可删除对应玩家对你领地的权限！

#### 指定玩家权限设置
如果你想对指定玩家设置在你领地里的权限，可以在领地里使用例如/res pset [玩家名] move false的命令来设置（将move替换成其他的即可）。

#### 领地信息查看
你也可以在别人的领地内使用/res current来查看领地的信息及其所有者，或在任意地方使用/res info [领地名]也行也可以用其查看你对其他玩家设置的权限限制！

#### 子领地的创建
如果你不是很信任某个人，但是你想和他住在一个领地里，那你可以使用子领地，也就是领地内的领地，在你的领地内使用/res subzone [子领地名]即可创建子领地，你可以对子领地设置不同的权限来达到不同的效果，例如给别人在子领地区域内的权限，但是不让其访问其他区域。

#### 领地的扩展
当你富有了并且你想扩展你的领地，很简单，只需要站在你的主领地内（不是子领地）然后看向你想扩展的方向后输入/res expand [数量]即可，数量指的是你想对着那个方向扩展多少格，成功后你的领地会扩展并更新它的保护须臾并且你也会花费对应的K币来达到此目的。如果想要缩小你的领地的话也一样，除了输入的指令变为/res contract [数量]罢了。

#### 重命名领地
重命名领地只需使用/res rename [领地旧名称] [领地新名称]即可！

#### 传送到指定领地
使用/res tp [领地名]即可传送到对应领地（不是你的也行，只要别人开放了tp权限）

#### 查看领地列表
使用/res list则可以查看你拥有的领地，而使用/res listall则可以查看服务器上有的所有领地（管理员）。

#### 删除领地
删除领地则须使用/res remove [领地名]，这样会删除对应领地名的领地（前提是他是你的或你有权限），如果想删除你有的所有领地，只需使用/res removeall即可。
